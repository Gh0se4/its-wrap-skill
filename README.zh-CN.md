# Its Wrap Skill

Its Wrap 是一个面向研究人员的 Skill，用来把粗糙的研究产出整理成产品决策材料。

## 它是什么

它帮助研究人员把未完成的工作翻译成一份 PM 可以继续判断下一步的交付包。

## 最适合的输入

- demo、原型、算法、prompt、笔记
- vibe-coding 项目
- 可运行的前端 / 后端 / API / 数据库
- AI / LLM 实验
- 还没有完全验证的早期成果

## 它覆盖什么

- 用户问题和场景
- 产品价值和产品形态
- 证据、指标和代理指标
- 替代方案与取舍
- 落地路径与推进阶段
- 风险、边界与缺失信息
- 关键问题不足时的澄清

## 它不做什么

- 不代替 PM 做最终判断
- 不做深度代码审查
- 不把所有输入强行套成同一模板
- 不隐藏未知项
