## Versioning Guide

Given a version number MAJOR.MINOR.PATCH:

1. **MAJOR** — Incompatible changes to output structure or core behavior
2. **MINOR** — New features, sections, or capabilities (backwards compatible)
3. **PATCH** — Bug fixes, clarifications, documentation improvements
